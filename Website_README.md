## üÜò TROUBLESHOOTING

**Website not showing after upload?**
- Make sure homepage is named `index.html` (lowercase)
- Check that files are in the `public_html` folder
- Wait 5-10 minutes for DNS propagation
- Clear your browser cache (Ctrl+F5)

**Forms not working?**
- Forms need backend processing (see Contact Form Setup section)
- Test forms send alert messages until connected to email service

**Mobile view looks weird?**
- Make sure you didn't change the `<meta viewport>` tag
- Test on actual device, not just browser resize
- Try clearing mobile browser cache

**Images not loading?**
- Current images use unsplash.com links
- Replace with your own uploaded images
- Make sure image file names match HTML code

**Need Help?**
- Contact your web hosting support
- Hire a local web developer
- Use Fiverr or Upwork for affordable help
